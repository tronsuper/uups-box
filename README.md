# UUPS TronBox Example

A simple TronBox project demonstrating the OpenZeppelin UUPS proxy pattern with upgradeable smart contracts.

## Installation

To use this box, run:

```bash
tronbox unbox uups
```

## Usage

Compile the contracts:

```bash
tronbox compile
```

Migrate (deploy) the contracts to the configured network:

```bash
tronbox migrate
```

Run tests:

```bash
tronbox test
```
